# Wordtrainer
A small Java application designed to help users practice vocabulary through an interactive and customizable training experience. The project focuses on clean OOP structure, responsive GUI design, and reliable file handling.

## Features

Vocabulary training with a responsive GUI:
Users can practice words through an intuitive and simple interface.

Customizable training sessions:
You can choose how many words you want to train in each session.

Save and continue later: 
The current training progress can be saved and restored, allowing you to continue your session at any time.

Input validation:
Ensures that user input is handled safely and correctly.

File management:
Words and game states are stored and loaded from files.
