# CurrencyConverter

A simple and intuitive Python application that converts amounts between different currencies using up‑to‑date exchange rates. The project demonstrates clean program structure, user‑friendly interaction, and practical API usage.

##  Features
Real‑time currency conversion:
Fetches current exchange rates from an online API.

Single or multiple currency queries: 
Users can request the exchange rate for one currency or several currencies at once, making the tool flexible for different use cases.

User‑friendly interface:
Simple input prompts guide the user through the conversion process.

Error handling:
Detects invalid currency codes, missing input, and connection issues.

Modular code structure:
Functions are separated clearly for readability and maintainability.

## Technologies & Concepts
Python

API requests

JSON parsing

Input validation

Modular programming

## Purpose
This project demonstrates how to interact with external APIs, process real‑time data, and build a clean, maintainable Python application that solves a practical everyday problem.
